# Restaurant_Sales_Data_Analysis_Mini_Project
Restaurant Sales Data Analysis mini project using Python, Pandas, Matplotlib, and Seaborn. The project explores sales performance, category-wise revenue, customer behavior, payment methods, and time-based trends to derive meaningful business insights from transactional restaurant data.

## Project Overview

This project analyzes restaurant sales data to extract meaningful business insights using **Python** and presents them through a structured **Excel dashboard**. The analysis focuses on sales performance, customer behavior, item popularity, payment methods, and time-based trends.

This project is suitable as a **data analytics portfolio project** demonstrating end-to-end analysis and dashboarding skills.

## Project Workflow

### 1. Data Loading

* Imported the CSV dataset into Python using Pandas

### 2. Data Cleaning

* Handled missing values and inconsistencies
* Converted columns to appropriate data types
* Prepared a clean, analysis-ready dataset

### 3. Exploratory Data Analysis (EDA)

* Calculated key metrics such as total sales, total orders, total customers, and average order value
* Performed category-wise, item-wise, and payment method analysis
* Conducted time-based analysis by day, month, and year
* Analyzed the relationship between quantity and order value

### 4. Data Export

* Exported the cleaned dataset to CSV for Excel-based dashboard creation

### 5. Excel Dashboard

* Built KPI cards for business metrics
* Created interactive charts for sales trends and performance comparison
* Added slicers for dynamic filtering
* Designed a clean, professional dashboard layout

## Key Insights Summary

* Main Dishes generate the highest revenue among all categories
* Sales remain relatively consistent across weekdays with minor variations
* Digital Wallets and Credit Cards together account for most transactions
* Quantity and Order Total show a moderate positive correlation
* Sales performance across 2022 and 2023 remains stable
